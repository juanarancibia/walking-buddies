# Walking buddies

Connect your wallet and add your walking buddies!

### Local execution

- Start ganache (port 7545)
- Move to directory `ui/contracts`
- Run `npm install`
- Run `truffle migrate`
- Move to directory `ui`
- Run `npm install` and then `npm run dev`
- Open **localhost:3000**

### Steps for use

- Link your wallet using Metamask
- Mint some buddies!
