# Walking buddies

Connect your wallet and add your walking buddies!

### Local execution

- Start ganache (port 7545)
- Move to directory `contracts`
- Run `npm install`
- Run `truffle migrate`
- Open [Walking buddies - Webpage](https://walking-buddies.vercel.app/)

### Steps for use

- Link your wallet using Metamask
- Mint some buddies!
